# Beanz

Beanz is a small-batch coffee storefront built with Astro and Tailwind. The project is tuned for static delivery, minimal client-side JavaScript, and strong Lighthouse scores.

## Highlights

- Zero-JS-first Astro build
- Responsive ecommerce-style marketing pages
- Optimized imagery and SEO-ready metadata

## Stack

- Astro
- Tailwind CSS
- Bun

## Quick start

```bash
bun install
bun run dev
```

Production build:

```bash
bun run build
bun run preview
```
